## PHASE 3: GENERATE 5 VISUAL HOOKS PER SCRIPT

For EACH script, generate 5 visual hooks. Mark one as ⭐ RECOMMENDED.

```
### ⭐ RECOMMENDED — VISUAL HOOK 1: "[NAME]"

**Pattern Interrupt Type:** [Physical Shock / Emotional Face / Motion+Face / Audio Jarring / The Stare]

[0:00-0:01]
VISUAL: [Face close-up + Motion entering frame]
FACE: [Emotion — defeat/shock/terror/etc.]
MOTION: [What moves — phone slam/calculator/dust/etc.]

[0:01-0:02]
VISUAL: [Reaction + Zoom into eyes]
FACE: [Physical signs — wet eyes/jaw drop/etc.]

[0:02-0:03]
TEXT OVERLAY: "[Specific numbers + Emoji]"
AUDIO: [Sound design — slam → silence → music]

**VIDEOGRAPHER:**
- Camera setup (lens, distance, angle)
- Lighting setup
- Actor coaching tip (what emotion to hit)

**EDITOR:**
- Timing (slow-mo speed, freeze frames)
- Effects (zoom, shake, color grade)
- Audio layers (sound effects + music cue)
```

### HOOK RULES:
1. Pattern interrupt in first 1 second
2. Face fills 70%+ of frame in close-ups
3. Motion and face reaction in SAME shot (no cut between them)
4. Text overlay appears at 0:02-0:03 (not earlier)
5. Use specific numbers in text (₹56K, 87 days, ₹33/day, etc.)
6. Emoji used strategically (max 2)
7. 3 seconds MAXIMUM for entire hook
8. Include videographer + editor instructions
