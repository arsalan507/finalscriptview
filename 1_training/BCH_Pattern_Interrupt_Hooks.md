# BCH PATTERN INTERRUPT HOOKS
## 98% Scroll-Stop Hooks | First 3 Seconds Only

**What is a Pattern Interrupt?**
A visual/audio element that BREAKS the viewer's expected scroll experience. Their brain says "wait, that's not normal" and forces them to stop and process.

**Pattern Interrupt Types:**
1. **PHYSICAL SHOCK** ‚Äî Unexpected action (throwing, breaking, falling)
2. **EMOTIONAL BREAK** ‚Äî Unexpected emotion (father crying, kid serious)
3. **VISUAL CONFUSION** ‚Äî Something in wrong place/context
4. **AUDIO JARRING** ‚Äî Silence‚ÜíLoud or Loud‚ÜíSilence
5. **DIRECT CONFRONTATION** ‚Äî Staring at camera, accusing viewer
6. **MOTION BREAK** ‚Äî Sudden freeze, slow-mo, or speed change

# MASTER PATTERN INTERRUPT TECHNIQUES

## TECHNIQUE 1: THE STARE
```
Actor stares DIRECTLY at camera for 2-3 seconds.
No movement. No blinking. No expression.
Creates discomfort ‚Üí Forces viewer to stop

WHEN TO USE: Serious/emotional hooks
```

## TECHNIQUE 2: THE THROW/DROP
```
Object thrown or dropped in slow-motion.
Brain anticipates impact ‚Üí Can't look away

WHEN TO USE: Frustration/anger hooks
```

## TECHNIQUE 3: THE DESTRUCTION
```
Something being destroyed (torn, burned, broken).
Primal reaction: "That's wrong" ‚Üí Forces stop

WHEN TO USE: Money/value hooks
```

## TECHNIQUE 4: THE FREEZE
```
Normal video suddenly FREEZES.
Brain says "error?" ‚Üí Attention captured

WHEN TO USE: Before-after or revelation hooks
```

## TECHNIQUE 5: THE AUDIO BREAK
```
Loud ‚Üí Complete silence (or vice versa)
Audio pattern interrupt = physical reaction

WHEN TO USE: Any hook (layer on top)
```

## TECHNIQUE 6: THE CLOSE-UP
```
Extreme close-up of unexpected detail.
Hands, eyes, dust, texture, numbers.
Forces viewer to process: "What is this?"

WHEN TO USE: Emotional/detail-focused hooks
```
