# MASTER HOOK FORMULAS

## Formula 1: THE CONTRAST HOOK
```
Split screen: PROBLEM vs SOLUTION
Text: Numbers + Emotion + "üëá"
Audio: Bad sound left, good sound right
```

## Formula 2: THE HANDS HOOK (Hathoda Style)
```
Extreme close-up of hands doing something
Slowly reveal context
Text: Emotional story tease
Audio: Ambient + music sting
```

## Formula 3: THE SHOCKING STAT HOOK
```
Text overlay with unexpected number
Visual that proves/shows the number
"Why?" or "How?" to create curiosity
Audio: Dramatic pause then music
```

## Formula 4: THE NIGHTMARE HOOK
```
Show worst-case scenario in progress
Character's distressed face visible
Text: Specific details (‚Çπ, KM, Days)
Audio: Tension sounds
```

## Formula 5: THE ROLE REVERSAL HOOK
```
Unexpected person doing unexpected thing
(Ajji riding, Kid teaching Appa, etc.)
Text: "X said no. Now X wants one."
Audio: Surprise/comedy sound
```
